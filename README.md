# FuelControl - Gestão de Combustível

Sistema completo para controle de vendas de combustível, gerenciamento de estoque de reservatórios e acompanhamento financeiro de forma intuitiva e segura.

## 🚀 Tecnologias Integradas

- **Frontend:** React 18, Vite, TypeScript, Tailwind CSS, shadcn/ui, Recharts.
- **Backend:** Node.js, Express, PostgreSQL (via Supabase).
- **Segurança:** Autenticação com senhas criptografadas (bcrypt).

## 🛠 Como executar o projeto localmente

1. Tenha o `Node.js` instalado.
2. Clone o repositório.
3. Instale as dependências:
   ```bash
   npm install
   ```
4. Suba o servidor de desenvolvimento (Frontend + Backend juntos):
   ```bash
   npm run dev:all
   ```
5. O sistema estará rodando em `http://localhost:8081`.

Para expor o servidor ao vivo e seguro para testes com a internet, você pode rodar o tunnel Cloudflare pelo terminal local fornecido no projeto.
